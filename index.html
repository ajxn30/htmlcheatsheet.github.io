Document Elements

Tag Name Description
[html] html tag Represents the root element.
[head] head tag Contains metadata about the document like title, style sheets and scripts.
[title] title tag Defines a title for the document and very important for SEO.
[body] body tag Repersents the content of the HTML document.

Content Sectioning

Tag Name Description
[header] header tag Repersents a container for introductory content or a set of navigational links.
[h1-h6] heading tag Repersents the six levels of headings. h1 is the highest one and h6 is the lowest.
[main] main tag Specifies the main contents of a document.
[nav]] nav tag Defines a set of navigational links, directing users within document or another page.
[section] section tag Repersents a standalone section.
[aside] aside tag Repersents a portion of a document whose content is indirectly related to the document's main content. .
[address] address tag Defines contact information.
[footer] footer tag Contains information about the author of the section, copyright data or links to related documents.

Texting Sectioning

Tag Name Description
[div] div tag Used as a container for HTML elements.
[dl & dd] dl & dd tag dl: Defines a description list. dd: describe a terminame in a description list.
[ul] ul tag unordered list of items. bulleted list.
[ol] ol tag ordered list of items. numbered list.
[li] li tag Repersents an item in a list.
[p] p tag Repersents a paragraph.
